# To-Do List Sederhana

Proyek aplikasi to-do list sederhana menggunakan PHP + CSS + JS.

Instalasi & menjalankan

1. Pastikan XAMPP (atau webserver PHP) terpasang.
2. Tempatkan folder `to-do-list` di `htdocs` (sudah berada di `d:/xampp/htdocs/to-do-list`).
3. Buka browser ke:

```
http://localhost/to-do-list/
```

Fitur
- Menambah tugas
- Tandai selesai/batal
- Hapus tugas

Data disimpan di `data/tasks.json` (file JSON sederhana).
