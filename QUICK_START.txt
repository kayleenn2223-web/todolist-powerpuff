================================================================================
TROUBLESHOOTING
================================================================================

Q: "404 Not Found" or page not found
A: 
  • Check folder path: F:\xampp\htdocs\todolist-powerpuff\
  • Access with: http://localhost/todolist-powerpuff/ (not with file:// or .php)
  • Make sure Apache is running in XAMPP

Q: "Folder data tidak ditemukan" or storage error
A:
  • App will auto-create "data" folder
  • If error: manually create "data" folder
  • Set folder permissions to 755
  • Make sure Apache user can write to folder

Q: "Email sudah terdaftar"
A:
  • Email must be unique per account
  • Use different email for each user
  • Can't register twice with same email
  • Use Forgot Password if you need recovery

Q: "Password tidak cocok" at registration
A:
  • Passwords must match exactly
  • Passwords are case-sensitive
  • Check CAPS LOCK
  • No spaces at beginning or end
  • Use 6+ characters minimum

Q: Tasks not saving
A:
  • Refresh page to see updates
  • Check browser console (F12) for errors
  • Verify "data" folder has write permissions
  • Check internet connection

Q: Can't logout or session expires
A:
  • Browser cache might be corrupt
  • Clear browser cookies
  • Try different browser
  • Restart browser completely

API ENDPOINTS (for developers)
================================================================================

Adding tasks via AJAX:
  POST /index.php
  Data: action=add&text="Task description"

Toggling task status:
  POST /index.php
  Data: action=toggle&id="task_id"

Deleting tasks:
  POST /index.php
  Data: action=delete&id="task_id"

Getting all tasks:
  GET /index.php
  Headers: X-Requested-With: XMLHttpRequest

Response format (JSON):
  [
    {
      "id": "unique_task_id",
      "text": "Task description",
      "completed": false,
      "created_at": 1234567890
    }
  ]
