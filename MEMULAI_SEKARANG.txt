================================================================================
                        LANGKAH-LANGKAH MEMULAI
================================================================================

STEP 1️⃣ : VERIFIKASI INSTALASI
================================

Pastikan folder berada di lokasi yang benar:
  📁 F:\xampp\htdocs\todolist-powerpuff\

Jika belum, pindahkan folder ke lokasi tersebut.


STEP 2️⃣ : JALANKAN APACHE
============================

1. Buka XAMPP Control Panel
2. Cari Apache di list
3. Klik tombol "Start"
4. Tunggu sampai statusnya hijau dan tertulis "Running"


STEP 3️⃣ : VERIFIKASI SISTEM (OPSIONAL)
===========================================

Buka browser dan akses:
  🔗 http://localhost/todolist-powerpuff/SYSTEM_CHECK.php

Semua check harus menunjukkan ✓

Jika ada error, ikuti instruksi yang diberikan.


STEP 4️⃣ : JALANKAN APLIKASI
==============================

Buka browser dan akses:
  🔗 http://localhost/todolist-powerpuff/

Anda akan diarahkan secara otomatis ke halaman Login.


STEP 5️⃣ : DAFTAR AKUN BARU
=============================

1. Klik tombol "Daftar sekarang" di halaman Login
2. Isi formulir dengan data Anda:
   
   📝 Nama Lengkap: (minimal 3 karakter)
      Contoh: Ahmad Wijaya
   
   📧 Email: (harus format valid)
      Contoh: ahmad.wijaya@example.com
   
   🔐 Password: (minimal 6 karakter)
      Contoh: MySecure123!
      → Lihat indikator kekuatan password:
         🔴 Lemah (Password terlalu sederhana)
         🟠 Sedang (Password lumayan kuat)
         🔵 Kuat (Password sudah cukup kuat)
         🟢 Sangat Kuat (Password sangat aman)
   
   ✓ Konfirmasi Password: (harus sama persis dengan password di atas)

3. Klik tombol "Daftar"

4. Jika berhasil, Anda akan melihat pesan sukses dengan link untuk login


STEP 6️⃣ : LOGIN KE APLIKASI
==============================

1. Masukkan email Anda
2. Masukkan password Anda
3. Opsional: Centang "Ingat saya" untuk tetap login
4. Klik tombol "Login"

Anda akan diarahkan ke Dashboard utama.


STEP 7️⃣ : GUNAKAN APLIKASI
=============================

Saat berada di Dashboard, Anda bisa:

✅ TAMBAH TUGAS:
   • Ketik deskripsi tugas di form input
   • Tekan Enter atau klik tombol "Tambah"
   • Tugas akan ditambahkan ke daftar

✅ TANDAI TUGAS SELESAI:
   • Klik tombol ✓ (centang) di sebelah tugas
   • Tugas akan dicoret (strikethrough)
   • Klik lagi untuk membatalkan

✅ HAPUS TUGAS:
   • Klik tombol ✕ (silang) di sebelah tugas
   • Tugas akan dihapus dari daftar

✅ KELUAR (LOGOUT):
   • Klik tombol "Logout" di navbar atas
   • Anda akan kembali ke halaman login


================================================================================
                          TIPS PENTING
================================================================================

💡 PASSWORD YANG KUAT:
   • Gunakan kombinasi huruf besar & kecil
   • Tambahkan angka
   • Tambahkan simbol (!@#$%^&*)
   • Minimal 6 karakter (lebih panjang lebih baik)
   • Jangan gunakan password yang mudah ditebak

💡 RESPONSIVITAS:
   • Aplikasi bekerja sempurna di smartphone
   • Coba buka di berbagai ukuran layar
   • Tombol dan form otomatis menyesuaikan

💡 SINKRONISASI DATA:
   • Semua tugas Anda tersimpan aman
   • Login dari device berbeda, tugas tetap ada
   • Data tidak akan hilang

💡 KEYBOARD SHORTCUTS:
   • Tekan Enter setelah mengetik tugas untuk submit
   • ESC untuk membatalkan input
   • Tab untuk navigasi form


================================================================================
                       JIKA LUPA PASSWORD
================================================================================

1. Di halaman Login, klik link "Lupa password?"
2. Masukkan email yang Anda daftarkan
3. Ikuti instruksi yang diberikan
4. Reset password Anda
5. Login dengan password yang baru

================================================================================
                        MASALAH UMUM & SOLUSI
================================================================================

❌ LOGIN GAGAL "Email atau password salah"
✅ Solusi:
   • Pastikan email benar
   • Pastikan password benar (case-sensitive!)
   • Periksa CAPS LOCK
   • Coba registrasi akun baru jika lupa

❌ ERROR "Email sudah terdaftar"
✅ Solusi:
   • Email harus unik untuk setiap akun
   • Gunakan email berbeda
   • Jika lupa email, gunakan forgot password

❌ HALAMAN TIDAK DITEMUKAN (404)
✅ Solusi:
   • Pastikan Apache sedang berjalan
   • Pastikan folder di lokasi benar
   • Akses dengan URL: http://localhost/todolist-powerpuff/
   • Jangan akses file .php langsung

❌ DATA TIDAK TERSIMPAN
✅ Solusi:
   • Refresh halaman
   • Periksa koneksi internet
   • Pastikan folder "data" ada dan writable
   • Cek browser console (F12) untuk error

❌ FOLDER DATA TIDAK ADA
✅ Solusi:
   • Aplikasi akan membuat otomatis
   • Jika gagal, buat manual folder "data"
   • Set permissions ke 755
